# myMovies

This is an independent project work of Developer Bootcamp 2022 organized by Saranen Consulting. The project is a movie themed single page app and stack is PostgreSQL, ASP.NET Core, ReactJS. This application is deployed on Heroku cloud service and can be run by clicking [here](https://mymoviesapplication.herokuapp.com/).
